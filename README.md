# Xcalcy
A Calendar of ruby-xcal for Conky<br>
注意：既に Conkyを活用している人は、<br>
~/.conkyrc が、上書きされるので注意してください。<br>
<br>
Ruby の xcal を、Conky で綺麗に表示するためのパッケージです。<br>
<br>
インストールするだけで日本語のカレンダーが表示されます。<br>
インストール方法は、<br>
<br>
sudo dpkg -i xcal-for-conky-1.0.0.deb<br>
sudo apt update<br>
sudo apt install --fix-broken<br>
これで、ログインし直すだけです。<br>
データを Googleから取得しているので、<br>
少し待ち時間を入れて表示されます。<br>
ネットの調子が悪いと表示されないので、ネット診断にもなります。<br>
お楽しみください。
